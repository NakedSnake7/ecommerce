# 🛒 Nombre del Proyecto

Sistema de tienda en línea desarrollado con Spring Boot.

## 🚀 Tecnologías

* Java 8+
* Spring Boot
* Thymeleaf
* MySQL
* Bootstrap

## ⚙️ Instalación

```bash
git clone https://github.com/tu-usuario/tu-repo.git
cd tu-repo
```

Configura tu base de datos en:

```
application.properties
```

Ejecuta el proyecto:

```bash
mvn spring-boot:run
```

## 🔐 Variables importantes

No subir credenciales reales.

## 📦 Funcionalidades

* Catálogo de productos
* Carrito de compras
* Panel administrativo
* Gestión de usuarios
